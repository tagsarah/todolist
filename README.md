# 📝 To-Do List Project

This is a simple **To-Do List App** built using **HTML, CSS, and JavaScript**.  
It allows users to add, mark, and delete tasks easily — helping them stay organized and productive. 🌟

## 💡 Features
- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- ❌ Delete tasks  
- 💾 Saves tasks in the browser (using localStorage)  
- 🎨 Simple and clean interface  

## 🧠 What I Learned
- DOM manipulation with JavaScript  
- Handling events (clicks, checkbox changes, etc.)  
- Working with Git, GitHub, and VS Code  
- Basic front-end design and layout
